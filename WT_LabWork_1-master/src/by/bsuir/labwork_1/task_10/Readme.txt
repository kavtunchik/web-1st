Как запустить программу на java из командной строки:
Вариант 1(Нескомпилирован файл):
1. Открыть командную строку;
2. Прописать команду java;
3. Добавить к ней полный путь к файлу и сам файл.

Вариант 2(Скомпилированный файл):
1. Открыть командную строку;
2. Прописать команду java -classpath;
3. Добавить к ней путь до группы пакетов, в которой лежит файл, например D:\WT\Labwork_1\out\production\Labwork_1(файл лежит в папке out обычно);
4. Добавить к пути чере пробел название скомпилированного файла вместе с пакетом через точку, например, by.bsuir.labwork_1.task_9.data.Main(без расширения .class);
5. Нажать Enter
